# HYPRSETTINGS
a simple Control Panel designed for hyprland as well this project are for my own dotfiles but is opened for everyone

# OVERVIEW
This program helpfull to manage screen light wallpaper audio and for fun sysinfo

# INSTALL
1)
```
$ make prepare
```
2)
```
# sudo make install
```

# Arch based
```
$ makepkg -si
```
